# Ses Dosyaları

Bu klasöre oyun için gerekli ses efektlerini ekleyin.

## Gerekli Ses Dosyaları:

1. **dice.mp3** - Zar atma sesi
2. **money.mp3** - Para kazanma/kaybetme sesi
3. **buy.mp3** - Mülk satın alma sesi
4. **card.mp3** - Şans/Kamu Sandığı kartı çekme sesi
5. **jail.mp3** - Hapishane sesi
6. **achievement.mp3** - Başarım kazanma sesi
7. **auction.mp3** - Açık arttırma başlama sesi

## Ücretsiz Ses Kaynakları:

- **Freesound.org** - https://freesound.org
- **Mixkit.co** - https://mixkit.co/free-sound-effects/
- **Zapsplat.com** - https://www.zapsplat.com
- **Soundbible.com** - http://soundbible.com

## Not:

Ses dosyalarını eklemezseniz, oyun sessiz çalışacaktır. Console'da "Sound play failed" hatası görebilirsiniz ama bu normal ve oyunu etkilemez.

## Örnek Ses Tipleri:

- **dice.mp3**: Zar yuvarlanma sesi, tavla taşı sesi
- **money.mp3**: Para sayma, nakit register sesi
- **buy.mp3**: Başarı "ding" sesi, satın alma tık sesi
- **card.mp3**: Kağıt karıştırma, kart çekme sesi
- **jail.mp3**: Kapı kapanma, hücre sesi
- **achievement.mp3**: Başarı fanfar, kutlama sesi
- **auction.mp3**: Çekiç sesi, ihale başlama sesi
